# 中文插件
首次安装 vscode 时，默认是英文，要切换中文，需要安装 `Chinese (Simplified)` 插件。

# 主题插件
One Dark Pro 主题，快捷键 `Ctrl+K Ctrl+T` 快速切换，选择浏览其他主题，选择 One Dark Pro Flat 或其他喜欢的主题。

# Error Lens
错误提示插件，它能将错误信息直接显示在代码行中。

# indent-rainbow
缩进彩虹插件，能将缩进颜色进行区分。

快捷键 `ctrl+shift+p` 打开命令面板输入：`user setting.json` 加入以下内容：
```json
"indentRainbow.lightIndicatorStyleLineWidth": 4,
    "indentRainbow.tabmixColor": "rgba(128,32,96,0.6)",
    "indentRainbow.indicatorStyle": "light",
    "indentRainbow.colors": [
        "rgba(255,255,64,0.2)",
        "rgba(127,255,127,0.2)",
        "rgba(255,127,255,0.2)",
        "rgba(79,236,236,0.2)"
    ]
```
