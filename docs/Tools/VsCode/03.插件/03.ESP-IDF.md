# 简介

> [!TIP]ESP-IDF 是 Espressif Systems 的一个开源项目，用于开发基于 ESP32 的物联网设备。

很多人说乐鑫的IDF环境配置很麻烦，其实并不那么麻烦。只需要简单理清几个步骤即可。
安装方式有离线和在线安装，在线安装可在 `vscode` 安装了`esp-idf` 插件后，按提示安装即可，这个方式虽然更便捷，但可能需要依赖网络环境才会比较顺畅。我个人不喜欢这种方式，所以会采取离线安装，看似更麻烦，但自由度更高。

视频教程可以参考 [正点原子的ESP-IDF环境搭建教程](https://www.bilibili.com/video/BV1EPisBWEUX?spm_id_from=333.788.videopod.episodes&vd_source=0fd232c2451c1b71f72acb39c0c2d7bf&p=4)。
# 下载离线包
安装需要先下载 [官方提供的离线包](https://dl.espressif.cn/dl/esp-idf)。\
本教程下载的是 `v5.4.3` 。理论上各版本间安装方式并无差别。

# 安装

1. 下载完成后，双击安装包即可。
2. 在系统检查的步骤中，如果应用修复按钮呈亮起状态，有可能是系统未支持长路径，先点击修复。
3. 待检查完成后，执行下一步操作。
4. 我习惯将安装路径设置为 `D:\software\sdk\Espressif` 中，若有需要可自行修改。
5. 剩下按默认步骤安装即可。
> [!CAUTION] 离线安装时，安装路径不能有中文，可能会导致安装失败。
> 这个安装过程耗时较长，请耐心等待。
> 安装完成后可能会弹出两个 cmd 和 PowerShell 的窗口，等待窗口执行完毕后，再关闭。

# 环境变量

windows系统下，快捷键：`win+R` 运行 `sysdm.cpl` 。
在 `高级` 选项卡下，点击 `环境变量` 按钮。
检查用户变量中是否存在这两个变量，如果没有就手动添加（具体的路径以自己的为准）：
表格

| 变量名 | 值 |
| :----- | :----- |
| IDF_PATH | D:\software\sdk\Espressif\frameworks\esp-idf-v5.4.3 |
| IDF_TOOLS_PATH | D:\software\sdk\Espressif |

改完后，点击 `确定` 退出。

# 连通vscode
在弹出的 ESP-IDF `Weclome` 窗口中，点击设置，即 `Configure extension` 继续。\
设置界面中选择 `USE EXISTING SETUP` ,选择我们手动安装的IDF。

# 新建项目

新建项目之前，需要一个模板。
1. 打开 `...\Espressif\frameworks\esp-idf-v5.4.3\tools\templates` 
2. 复制 `sample_project` 文件夹。
3. 粘贴到：`D:\software\sdk\Espressif\frameworks\esp-idf-v5.4.3\examples\get-started`

> [!NOTE] 这么做的目的是为了今后创建项目时，不需要从0创建。而这个示例文件夹的配置和文件是最干净的。

1. 创建一个空的项目文件夹，用 `vscode` 打开。
2. 快捷键：`ctrl+shift+p` 输入 `ESP-IDF: 新建项目` 来创建项目（右下角如果有激活提示，点激活）。
3. 选择安装的版本，比如上面安装的是 `v5.4.3` ，这里就选择 `v5.4.3` 。
> [!NOTE] 如果电脑中有多个版本，会显示多个选项，选择自己需要的版本即可。
1. 在弹出的窗口中，展开 `ESP-IDF Examples` => `get-started` ，双击 `sample_project` 文件夹，点击 `Create...project` 创建项目。
2. 输入项目名称、项目路径、选择设备型号、其他默认，点击 `Create Project` 创建项目。
3. 创建完成后，点击 `Open Project` 打开项目。

至此，IDF环境搭建 => 项目新建完成。
